# IOSignal Monorepo

This monorepo contains the core packages for IOSignal, a real-time communication library.

## Packages in this Monorepo

| Package | Description | NPM |
|---|---|---|
| [`boho`](/packages/boho) | Secure Lightweight Encryption & Authentication Library | [![npm](https://img.shields.io/npm/v/boho.svg)](https://www.npmjs.com/package/boho) |
| [`iosignal`](/packages/iosignal) | Real-time communication library for Node.js, browsers, and Arduino | [![npm](https://img.shields.io/npm/v/iosignal.svg)](https://www.npmjs.com/package/iosignal) |
| [`iosignal-cli`](/packages/iosignal-cli) | CLI for IOSignal server and client | [![npm](https://img.shields.io/npm/v/iosignal-cli.svg)](https://www.npmjs.com/package/iosignal-cli) |
| [`meta-buffer-pack`](/packages/meta-buffer-pack) | Binary data serialization & deserialization with metadata | [![npm](https://img.shields.io/npm/v/meta-buffer-pack.svg)](https://www.npmjs.com/package/meta-buffer-pack) |

## Separately Managed Packages

-   [`iosignal-examples`](https://github.com/remocons/iosignal-examples): IOSignal usage examples.
-   [`iosignal-arduino`](https://github.com/remocons/iosignal-arduino): Arduino library for IOSignal.
-   [`remocon-arduino`](https://github.com/remocons/remocon-arduino): Arduino library for `remocon`.
-   [`boho-arduino`](https://github.com/remocons/boho-arduino): Arduino library for `boho`.
